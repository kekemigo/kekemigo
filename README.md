## 功能

- 词汇：雅思词汇真经内容，按场景分组。
- 音标：每个词下面显示 IPA 音标，方便跟读和记忆发音。
- 例句翻译：词汇表里的英文例句已补充中文翻译。
- 备注讲解：拓展栏补充例句生词、搭配、句式讲解。
- 同组辨析：对同一小组里意思相近、容易混淆的词补充区别说明。
- 单词练习：支持词汇打字练习。
- 语法：整理雅思语法课程相关内容。
- 听力：包含听力基本概念、179 考点词和雅思王语料库相关内容。
- 阅读：整理 538 考点词同义替换。
- 写作：包含 100 句翻译练习和大小作文内容。

## 词汇

词汇部分基于雅思词汇真经内容，包含核心词汇、IPA 音标、场景分组、原书音频、例句翻译和备注讲解。

备注栏会尽量帮助区分同组近义词，例如 disaster / calamity / mishap、gust / breeze / gale、hurricane / typhoon / tornado 等。
